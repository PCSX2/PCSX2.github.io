Nightly Builds

Nightly builds are less tested than a stable release and may contain bugs or regressions not yet fixed. However, they do contain the current up-to-date state of the project. If you need a recently added feature or fix, or want to want to assist in the testing process -- these are the builds for you!

Test Builds

These builds are in-progress changes yet to merge into PCSX2 and become part of a Nightly build. If you would like to assist in testing out these builds here is where you can get started.

  • You will need a GitHub account to successfully download these builds.
  • If you discover a problem, you should leave a comment explaining the problem and how to reproduce it on the Pull Request itself.
  • Lastly, if you think testing is for you, consider reaching out in the discord to become a more official tester!
